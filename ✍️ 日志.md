         ! Emmet 是一种用于提高编码效率的工具
         \! 它允许开发者使用缩写语法来生成 HTML 和 CSS 代码
     ! 常用语法
         ! 元素：直接输入元素名称即可生成对应的 HTML 元素
             ！ 如输入 div 即可生成 <div></div>
         ! 类名：在元素名称后面添加点号（.）和类名即可生成对应的类名
             ！ 如输入 div.class-name 即可生成 <div class="class-name"></div>
         ! ID 名：在元素名称后面添加井号（#）和 ID 名即可生成对应的 ID 名
             ！ 如输入 div#id-name 即可生成 <div id="id-name"></div>
         ! 子元素：在父元素后面添加大于号（>）和子元素名称即可生成对应的子元素
             ！ 如输入 div>p 即可生成 <div><p></p></div>
         ! 兄弟元素：在元素后面添加加号（+）和兄弟元素名称即可生成对应的兄弟元素

@ HTML
     ! 超文本标记语言
     ! 通过开头打标签来定义格式
         ？ 与 Markdown 语言的关系
             ! Markdown 语言是一种轻量级标记语言，用于在文本中添加格式化元素
             ! HTML 语言是一种用于创建网页的标记语言，用于定义网页的结构和内容
     ! 结构： 
         ! 文档类型声明（<!DOCTYPE html>）是 HTML 文档的第一行
         \! 用于告诉浏览器这是一个 HTML 文档
         ! html 标签（<html lang="zh-CN">）是文档的根元素，包含了所有其他元素
         ! head 标签（<head>）包含了文档的元数据，如标题、字符编码、视口设置等
         ! meta 标签（<meta charset="UTF-8">、<meta name="viewport"  
         \! content="width=device-width, initial-scale=1.0">）
         \! 用于设置字符编码和视口设置
         ! title 标签（<title>Project 01</title>）用于设置文档的标题
         ! body 标签（<body>）包含了文档的主体内容，如段落、标题、图片、链接等
         ! div 标签（<div>）用于定义文档中的一个区域，通常用于组织其他元素
             ! 可以使用 div 标签将多个元素组合在一起, 如段落、标题、图片等
             \! 并为其添加样式或功能，如设置背景颜色、边框、内边距等
         ! p 标签（<p>）用于定义段落
         ! span 标签（<span>）用于定义行内元素，通常用于对文本进行样式化或添加其他功能
         ! strong 标签（<strong>）用于定义加粗文本
         ! em 标签（<em>）用于定义斜体文本，通常用于强调文本
             ! 与 strong 标签不同，em 标签的文本通常会显示为斜体
         ! u 标签（<u>）用于定义下划线文本
         ! s 标签（<s>）用于定义删除线文本
         ! i 标签（<i>）用于定义斜体文本
         ! b 标签（<b>）用于定义加粗文本
         ! br 标签（<br>）用于换行
         ! hr 标签（<hr>）用于水平线
         ! img 标签（<img src="image.jpg" alt="图片描述">）用于插入图片
         ! a 标签（<a href="https://www.example.com">链接文本</a>）用于插入链接
         ! ul 标签（<ul>）用于定义无序列表
         ! ol 标签（<ol>）用于定义有序列表
         ! li 标签（<li>）用于定义列表项
         ! table 标签（<table>）用于定义表格
             ! caption 标签（<caption>）用于定义表格标题
             ! tr 标签（<tr>）用于定义表格行
             ! th 标签（<th>）用于定义表格标题单元格
             ! td 标签（<td>）用于定义表格单元格
