## 💾📃 笔记模板
```
T 时间
@ 实体
     ！ 事实、情况 
     \！ 表示非完全换行
     ～ 假设
         ！ 事实
             = 推论、结果
     ： 陈述
     ？ 问题
         + 尝试、行动
             ？* 未解决的问题
                 +* 未进行的尝试
```
